Proposed solution: Enter TStore

What is TStore

TStore is a Python library for flexible storage and processing of (spatial) TS data. Two key features:

  • TS üïê encapsulation: TS, TSDF, TSLong and TSWide objects to organize hetereogeneous (spatial) time series data into Python data frames
  • TS üïê storage: TStore is a hierarchically-structured specification to reliably and efficiently store (spatial) TS data based on Parquet (and GeoParquet).
